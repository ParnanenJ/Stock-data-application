## Tietoja sovelluksesta
Osaketietosovellus on sovellus, jolla käyttäjä pystyy tarkistamaan osakkeiden realiaikaisia tietoja (esim. erilaisia hintatietoja ja arvon kehitystä). Sovelluksessa on käytetty Financialmodelingprep:n ilmaista apia, 
jonka takia sovelluksen osakehakeu on rajattu ( etenkin hintatietoja hakiessa). Sovellus toimii siten että osakkeiden tietoja haetaan osakkeen lyhenteen avulla. Tämän jälkeen mikäli api pystyy hakemaan osakkeeseen 
liittyviä tietoja, näytetään ne käyttäjälle selkeässä käyttöliittymässä. Sovellus on myös responssiivinen eli sitä voi käyttää myös puhelimella.

## Teknologiat 
Käytin seuraavia teknologioita: `html`, `css`, `javascript` ja `bootstrap`.  
Teknologioita käytettiin seuraavasti:  
`html`: luotiin sovelluksen käyttöliittymän perusrakenne  
`bootstrap`: määritettiin tyylit käyttöliittymälle  
`css`: muokattiin tai tarkennettiin tyylejä  
`javascript`: lisättiin kaikki toiminnallisuus sovelluskelle
